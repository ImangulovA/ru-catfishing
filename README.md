# game-designer

Workspace for designing and building a small web game with a built-in social
loop (players can share their results, Wordle-style).

## Structure
- `research/` — notes on existing games we learn from (e.g. `catfishing.md`)
- `prototype/` — playable prototypes / the game itself
- `scripts/` — helper scripts (data fetching, generators, etc.)

## Status
- 2026-06-05: project created. First research target: catfishing.net (Kevan Davis).

## North star
Daily puzzle + tiny shareable result + zero login. Cheap to host
(ideally a static site on GitHub Pages with a daily seed, no backend).
